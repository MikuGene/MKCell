MKCell:  Cell milk shake  exploration.

MKCell serves as an analytical tool to identify cell types or cell components and their expression profiles in single-cell or bulk RNA-seq, respectively.

MKCell R package

On October 19, 2020 - we officially updated the MikuCell Github repository to release 1.0 !

MKCell tutorial

The detailed tutorials for using the MKCell analysis platform can be viewed on the help page.

Tutorial

Cell composition

MKCell allows to estimate the cell composition when you upload bulk RNA-seq data.

Cell recognition

MKCell allows to identify known cell phenotype when you upload single cell RNA-seq data.

Pan-cancer view

MKCell allows to view Pan-cancer data which has been analyzed, from TCGA, GEO, HCA, ect.

Custom analysis

MKCell allows to download data and provides some online-tools for custom analysis.
